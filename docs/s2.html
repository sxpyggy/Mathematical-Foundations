From the graph, we can get the expressions of the curves. TR is a quadratic equation about Q, TC is a first order equation about Q.You can substitute the value of the special points into the equation to solve the problem. \[ TR = -2Q^2+200Q \] \[ TC = 55Q+500 \] \[ \pi = TR-TC \\ = (-2Q^2+200Q) - (55Q+500) \\ = -2Q^2+145Q-500 \] When \[ Q=-\frac{b}{2a}\\ =\frac{145}{4} \] Substitute the value of Q into the equation to get the maximum profit. \[ \pi = -2\times (\frac{145}{4})^2+145\times \frac{145}{4}-500\\ = 2128.125\] It is closest to 2100, so we choose B.

Question 6

Answer : D

\[ Q=f(K,L) \] is said to be homogeneous if \[ f(\lambda K, \lambda L)=\lambda^n f(K,L) \] As \[ f(K, L)= 8 \sqrt{L^2K}+2KL \] \[ f(\lambda K, \lambda L)=8 \sqrt{(\lambda L)^2\lambda K}+2(\lambda K)(\lambda L) \\ =8 \lambda^\frac{3}{2} \sqrt{L^2K}+2 \lambda^2 KL\\ \neq\lambda^n f(K,L) \] Thus, it is not homogeneous, we choose D.

Question 9

Answer : A

(c)As \[ \pi = TR-TC=PQ-TC \] When there is decrease in fixed costs, TC is decreased, profits is increased. And the fixed costs only affects the constant of the quadratic curve, so the curve shifts vertically upwards.
